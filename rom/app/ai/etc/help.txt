## AI Bang Commands

### !exec - Run shell command
!exec <shell command>

Execute a shell command. Full shell support: pipes, redirects, globs, chaining.

Examples:
  !exec ls -la /bin
  !exec find /etc -name '*.txt' | head -5
  !exec cat /var/log/messages | grep error | wc -l

### !call - Direct syscall
!call <syscall> [args...]

Call a kernel syscall directly. Arguments are parsed as JSON or plain strings.

Syscall namespaces:
  file:*   - stat, readdir, mkdir, open, read, write, unlink, rename
  proc:*   - spawn, wait, exit, getpid, getcwd, chdir, getenv, setenv
  llm:*    - complete, chat, stream, embed
  ems:*    - query, create, update, delete
  auth:*   - login, token, grant
  port:*   - create, recv, close
  handle:* - send, close, redirect, restore

Examples:
  !call file:stat "/etc/hosts"
  !call file:readdir "/"
  !call proc:getpid
  !call proc:getcwd

### !ems - EMS database operations
!ems <subcommand> [args...]

Interact with EMS (Entity Management System) database tables.

Subcommands:
  describe [model]     - List all models or show schema for one
  select <model> ...   - Query records
  create <model> ...   - Create a record
  update <model> ...   - Update a record
  delete <model> ...   - Soft delete a record
  revert <model> ...   - Restore soft-deleted record
  expire <model> ...   - Permanently delete

Examples:
  !ems describe
  !ems describe ai.request
  !ems select ai.request limit 5
  !ems create ai.stm content="hello" salience=5

### !ref - Search memories
!ref <keywords...>

Search short-term and long-term memory by keywords.

Examples:
  !ref typescript error handling
  !ref user preferences

### !coalesce - Force memory consolidation
!coalesce

Immediately consolidate short-term memories into long-term storage.

### !spawn - Start async subagent
!spawn "task description"
!spawn {"task": "...", "model": "claude-haiku-3"}

Start a subagent to handle a task asynchronously. Returns a spawn ID.

Examples:
  !spawn "summarize the contents of /var/log"
  !spawn {"task": "count files", "model": "claude-haiku-3"}

### !wait - Wait for subagent result
!wait
!wait spawn:<id>

Wait for all subagents or a specific one to complete, then return results.

Examples:
  !wait
  !wait spawn:a1b2c3d4

### !help - Show this help
!help

Display available bang commands and usage.
